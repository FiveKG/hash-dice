游戏玩法：

1、游戏可选单行、双数、顺子、三同；

2、用户可购买数量从 0.1EOS-100EOS；

3、当开出的数字组合与用户购买的数字组合一致，则用户按相应的赔率获得奖金；

开奖方法：

开奖数字 = 用户购买的交易 ID 取数字后 3 位

交易ID：5eeb2b24e5f3219cb9f408079086210af7946c4b04dfeaa9a40dcb3313224cc4

如：

则开奖数字 = 244 = 双数

当用户购买的数字组合为 双数 时，则用户按相应的赔率获得奖金

分配方法：

1、用户中奖奖金 = 购买金额X赔率；

2、购买金额的 1% 进入 Bingo 分红池；

3、购买金额的 1% 按购买用户进行推荐佣金的分配；

直接推荐 # 1 获得购买额的 0.6%；

间接推荐 # 2 获得购买额的 0.3%；

间接推荐 # 3 获得购买额的 0.1%；

4、数字组合规则：

单行：三个数字中无法组合成双数、顺子、三同，如：371；

双数：三个数字中有两个相同且相邻的数字，如：355，有两个相同但不相邻属于单行，如：373；

顺子：三个数字中可以按顺序、倒序、乱序组合为顺子的数字，如：456，654，465；

三同：三个数字都相同，如：666。
