三三静态分配制度

 

1、所有参加排序的均为子账号，包含用户的投资子账号和复投子账号；

2、每个子账号下只能有3个子账号排列；

3、推荐的子账号在按规则有空位可排列层级里随机排列；

4、所有的静态分配由排列的位置决定，与直接推荐关系无关；

例：

 

如上图

1、A推荐的第1个子账号，在向下一层有空位，随机排列在第2位

2、A推荐的第2个子账号，在向下一层有空位，随机排列在第1位

3、这时A的上方推荐的子账号排列在第2位

4、A向下1层为3个账号已满

5、A推荐的第3个子账号，向下二层有空位，随机排列后在3的位置

6、A推荐的第4个子账号，向下二层有空位，随机排列后在4的位置

7、A复投产生的第2个子账号，向下二层有空位，随机排列后在5的位置
